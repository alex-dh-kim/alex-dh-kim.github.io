Project Epsilon

A personal blog platform with markdown support and dynamic routing.

Technologies Used

Next.jsMDXCSS Modules

Project Details

Project Epsilon is a personal blogging platform designed for developers and content creators. It supports writing posts in Markdown, which are then rendered dynamically. Features include tag-based filtering, search functionality, and a clean, minimalist design. Next.js's dynamic routing is utilized for SEO-friendly URLs and fast navigation.
